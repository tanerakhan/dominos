## Versiyonlar

    NodeJS          -   v10.20.0
    NPM             -   6.14.4

## Client

    Başlangıç       -   npm install && npm start

    Component       -   Küçük yapıların bulunduğu klasör
    Container       -   Sayfaların servis çağrılarının yapıldığı klasör
    Assets          -   Image ve scss'lerin olduğu assets klasörü
    Store           -   Redux'ın çalıştığı klasör

## Server

    Başlangıç       -   npm install && npm start

    Routes          -   all.js projenin tek router'ı ve buradan data.json içerisinde file system aracılığıyla okumalar yapılıyor.
    Not: Proje Public klasörü içerisinde data.json' dan besleniyor.
